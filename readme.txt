## General Information
- Users can genrate any size of pixel arts.
- Users can paint pixels using variaty of colors.
- Mobile friendly Responsive design ( from 8k screensize to mobile screensize).
- Responds to user clicks and long presses when drawing the pixel arts.

## Technologies Used
- HTML
- CSS 
- Javascript


## Features
- When the size is provided this web app will generate pixel boxes
- when the generate button is clicked new grid need to be generated with desired size
- app will always listen to the generate button click when it is clicked it will call for populate button
